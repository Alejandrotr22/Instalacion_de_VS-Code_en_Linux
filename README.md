# Instalacion_de_VS-Code_en_Linux
Este repositorio está destinado a la instalación del IDE de microsoft: Visual Studio Code

## Introducción

### ¿Qúe es Visual Studio Code?
  
Visual Studio Code es un editor de código fuente. Es compatible con varios lenguajes de programación y un conjunto de características que pueden o no estar disponibles para un lenguaje dado.

## Pasos

### 1.Descargar la versión estable de VS-Code

Primero tenemos que descargar la versión estable con este comando: 
```
sudo snap install --classic code
```

### 3.Otras extensiones muy recomendables


- Path Intellisense:

Una extensión que permite escribir rutas más fácilmente

<div align="center">
<img width="40%" src="http://drive.google.com/uc?export=view&id=1LZm5BHLnvoIGvMxMhVQDhf_7WpnfI-Zn">
</div>

- Bracket Pair Colorizer:

	Una extensión que ayuda a ver mejor los [] o {}
  
<div align="center">
<img width="40%" src="http://drive.google.com/uc?export=view&id=1dXAqKq3TKEp301MPAa2oT8QSFWlBbdUw">
</div>

- GitLens:
Sobrealimenta las capacidades de Git que ya se encuentran integradas en Visual Studio Code. Ayuda a visualizar el autor del código, navegar y explorar sin problemas los repositorios de Git, obtener información valiosa a través de potentes comandos de comparación y mucho más.

<div align="center">
<img width="40%" src="http://drive.google.com/uc?export=view&id=1IKmAXjTdLm38Eexd3wr9Slb7H2n8le0f">
</div>
	
- Prettier:

	Herramienta que formatea el código 	automáticamente, esto permite despreocuparse 	de si nuestro código esta bien identado.
  
 <div align="center">
<img width="40%" src="http://drive.google.com/uc?export=view&id=1OeUCjjujasJ3EJNQ70Yo2rPnDvcSo3MI">
</div>

- Color Highlight:

Facilita la visualización de los colores. Rodea el código hexadecimal del color en un rectángulo con el color elegido.
<div align="center">
<img width="40%" src="http://drive.google.com/uc?export=view&id=1iZwLEsldXZE2BnzTMkWj4ZhxkuRTv2qN">
</div>


- Indent Rainbow:

	Esta extensión colorea la sangría frente a su texto alternando cuatro colores diferentes en cada paso, ayuda a visualizar el correcto indentado del código.

<div align="center">
<img width="40%" src="http://drive.google.com/uc?export=view&id=1flldw-hGjwZEnH-2Dim_79itysO1dBza">
</div>
